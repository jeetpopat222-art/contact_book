<h2 align="center">☠️ WHAT IS THIS BEAST? ☠️</h2>

<p align="center">
A <b>terminal-based contact book</b> that lets you store, find, and destroy contacts.<br><br>
No fancy database. No internet. No nonsense.<br>
Just Python, a text file, and the will to survive.
</p>

<h2 align="center">🚀 HOW TO RUN THIS 🚀</h2>

**Step 1 — Clone it:**
```bash
git clone https://github.com/yourusername/contact-book.git
cd contact-book
```

**Step 2 — Run it:**
```bash
python contact_book.py
```

**Step 3 — You will see this:**
```
1. check contact
2. add contact
3. search contact
4. delete contact
enter your mode:
```

**Step 4 — Go crazy.**

<h2 align="center">📖 HOW CONTACTS ARE STORED 📖</h2>

No fancy database. Just a plain text file:

```
alice,1234567890
bob,9876543210
charlie,5555555555
```

Simple. Clean. Deadly.

---

<h2 align="center">⚠️ WARNINGS ⚠️</h2>

> 🔴 Deleting a contact is **permanent**. There is no undo.  
> 🔴 Don't delete `contact.txt` or your contacts are gone forever.  
> 🟡 Names are stored in **lowercase** — don't panic.  
> 🟢 The confirmation screen before delete will save your life.
